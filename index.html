What is a Binary Tree, a Binary Search Tree, a Strict Binary Search Tree?
All tree nodes have references to their parents and to their kids and contain data!

A Strict Binary Search Tree maintains the following properties:
  • It sorts the data when new nodes are added or when nodes are removed!

  • A Binary Search Tree maintains the following properties:

    • The in-order transversal is in ascending order in an order like alphabetic or numeric or alphanumeric, etc.

    • A Binary Tree maintains the following properties:
      • It has at most 1 parent node.
      • It has at most 2 child nodes.
      • It can have as few as 0 parent nodes. When this is the case, the parent node is null.
      • It can have as few as 0 child nodes. When this is the case, both of the kids are null.
